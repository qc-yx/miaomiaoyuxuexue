# 苗苗语学习游戏

一个基于Web的互动学习游戏应用，包含转盘游戏、美食推荐、运动提醒等功能。

## 功能特性

- 🎯 转盘游戏：自定义选项和主题
- 🍽️ 美食推荐：智能分类和历史记录
- 💪 运动提醒：个性化运动计划和提醒
- 📝 笔记管理：记录学习笔记和想法
- 👥 邀请系统：邀请码绑定和奖励机制

## 技术栈

- 前端：HTML5, CSS3, JavaScript (ES6+)
- 后端：Node.js, Express.js
- 数据库：PostgreSQL (Neon)
- 认证：JWT
- 部署：GitHub Pages

## 项目结构

```
├── frontend/          # 前端文件
├── backend/           # 后端代码
├── config/           # 配置文件
├── images/           # 图片资源
└── docs/            # 文档
```
